# Mystery Escape Game

![Room Screen](Images/Room-Screen.png)

## Overview

Mystery Escape Game is a Java-based text adventure where players must solve riddles to escape a locked room. The project was developed to strengthen object-oriented programming skills while creating an engaging puzzle-solving experience.


## Features

- Interactive UI
- ![Pause Screen](Images/Pause-Screen.png)
- Multiple riddles
![Riddle Screen](Images/Riddle-Screen.png)
- Player choice
![Main Menu](Images/Main-Menu.png)
- Player input validation
- Room animations
![Animation](Images/Animation-Screen.png)
- Modular object-oriented design
![Game Over](Images/Game-Over-Screen.png)

## Technologies Used

- Java
- Object-Oriented Programming
- libGDX
- Online assets for art
